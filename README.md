## About

A modern, responsive web application built with React and Vite that allows users to explore culinary options, browse menus, and visualize their next meal.

## Features

- Interactive Menu Browsing: Users can filter and view dishes by categories (Sandwiches and wraps)
- Dynamic UI: Fast, component-based user interface powered by React

## Tech Stack

- React
- Vite
- CSS

## Installation

1. Install dependencies:

```bash
npm install
```

## Run Locally

Start the development server:

```bash
npm run dev
```
